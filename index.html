First time you want to add your project then you need to use this liens of command.

1.echo "# web-course-repo" >> README.md

2.git init

3.git add .{if you want to add specific file then you can use file name with extention otherwise you use dot .}

4.git config --global user.email "your github email" if you need to config user email

5.git config --global user.name "your github user name"" if you need to config user name

6.git commit -m "first commit"

7.git branch -M main

8.git remote add origin https://github.com/aftabgroupbd/git-init-code.git

9.git push -u origin main

If you need to update code then you can use this command

1.git add .

2.git commit -m "your commit"

3.git push
