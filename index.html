সব থেকে বেশি ব্যাবহার গিট কমান্ড

কমান্ড ব্যবহার
git init লোকাল মেশিনে গিট শুরু করতে এই কমান্ড ব্যবহার করা হয়
git clone ssh://git@github.com/[username]/[repository-name].git আগের কোন রিপোজিটোরি লোকাল মেশিনে নিতে হলে এই কমান্ড ব্যবহার করা হয়
echo "# your-repository-name" >> README.md রিডমি ফাইল তৈরি করতে এই কমান্ড ব্যবহার করা হয়
git add [file-name.txt] কোন একটা ফাইল রিপোজিটোরিতে এড করতে এই কমান্ড ব্যবহার করা হয়
git add . সব ফাইল রিপোজিটোরিতে এড করতে এই কমান্ড ব্যবহার করা হয়
git config --global user.email "your github email" কনফিগারেশনে ইমেইল সেট করতে এই কমান্ড ব্যবহার করা হয়। এটা আপনার ডিভাইসে একবার করলেই হয়
git config --global user.name "your github user name" কনফিগারেশনে নাম সেট করতে এই কমান্ড ব্যবহার করা হয়। এটা আপনার ডিভাইসে একবার করলেই হয়
git commit -m "[commit message]" কমিট করতে এই কমান্ড ব্যবহার করা হয়
git branch -M branch-name[By default your branch-name is main] লোকাল ব্রান্সের নাম সেট করতে এই কমান্ড ব্যবহার করা হয়
git remote add origin https://github.com/user-name/repository-name.git যে রিপোজিটোরিতে এড করতে চান তার জন্যে এই কমান্ড ব্যবহার করা হয়
git push -u origin branch-name[By default your branch-name is main] অন্য ব্রান্সে পুশ করতে এই কমান্ড ব্যবহার করা হয়।
git push পুশ করতে এই কমান্ড ব্যবহার করা হয়।
git branch ব্রান্স লিস্ট দেখতে এই কমান্ড ব্যবহার করা হয়।
git branch [branch name] নতুন ব্রান্স তৈরি করতে এই কমান্ড ব্যবহার করা হয়।
git checkout [branch name] ব্রান্স পরিবর্তন করতে এই কমান্ড ব্যবহার করা হয়।
git checkout -b [branch name] নতুন ব্রান্স তৈরি সেই ব্রান্সে যেতে এই কমান্ড ব্যবহার করা হয়।
git merge [branch name] এখন যে ব্রান্সে আছেন তার সাথে অন্য ব্রান্স যোগ করতে এই কমান্ড ব্যবহার করা হয়।
git merge [source branch] [target branch] এক ব্রান্সকে অন্য ব্রান্সের সাথে যোগ করতে এই কমান্ড ব্যবহার করা হয়।
